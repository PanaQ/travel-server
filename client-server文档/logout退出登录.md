**简要描述** 
- logout退出登录

**版本信息**
- 1.0

**请求URL** 
- `/user/loginout `


**请求示例**

```JSON
{
    "reqId": "req1512973735226",
    "timestamp": 1512973735226,
    "param": null
}
```


 **返回示例**

```JSON
{
    "code": 0,
    "message": "ok",
    "reqId": "14343543543254",
    "result": null
}
```
